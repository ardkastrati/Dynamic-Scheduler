Changes the scheduling strategy. Returns the new schdeuling strategy with all tasks, or N\+U\+L\+L if there was an error.

Estimated runtime of tasks will get lost by changing from statistically based strategies (L\+S\+F, \hyperlink{a00035}{S\+J\+F}) to non-\/statistically based strategies. The order of the old queue will be kept.
