Detailed Description

The MpiWinSchedulingStrategy interface is a specialization of the SchedulingStrategy interface. The MpiWinSchedulingStrategy interface adds function for remote scheduling queue access. Implementations must provide the steal_next_task() and the get_task_count function on remote scheduling queues by given rank. The tasks must be kept in a MPI window to support remote memory access.
